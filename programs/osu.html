    Open Security Utilities, modular repository with programs for OpenSecurity

Open Security Utilities is modular repository with programs for OpenSecurity.

OSU supports all of thoses optional features:

  • Magnetic Card
  • Security Door
  • RFID Reader
  • RFID Card Reader
  • Keypad
  • Alarm
  • Card Writer
  • Energy Turret
  • Rolldoor
  • Entity Detector
  • Biometric Reader
  • Security Terminal
  • NanoFog
  • Network Integration
  • Redstone Integration
  • Redstone Card
